حتماً. اینجا یک **سیستم دیزاین کامل و حرفه‌ای برای یک وب‌سایت فارسی با تم رنگی Royal Blue، بدون هیچ بوردر، با استایل شیشه‌ای (Glassmorphism)** و با رعایت تضاد رنگ متن‌ها و پس‌زمینه‌ها برات نوشتم.

---

## 🎨 سیستم دیزاین وب‌سایت

**با تم رنگی: `Royal Blue` + استایل شیشه‌ای + بدون بوردر + متن‌های خوانا روی بک‌گراند متضاد**

---

### 1. 📌 فلسفه طراحی

* **Glassmorphism**: حس مدرن، ظریف و آینده‌نگرانه.
* **بدون بوردر**: به‌جای مرز، از سایه و تضاد رنگی استفاده می‌کنیم.
* **تضاد متون و پس‌زمینه**: خوانایی بالا در تمام اجزاء سایت.
* **فونت فارسی**: یک فونت خوانا، نرم و حرفه‌ای مثل [IRANSansX](https://fontiran.com/%d8%ae%d8%a7%d9%86%d9%88%d8%a7%d8%af%d9%87-%d9%81%d9%88%d9%86%d8%aa-iransansx/) یا **Shabnam**.
* **زبان سایت: فارسی (rtl)**

---

### 2. 🌈 پالت رنگ‌ها (بر اساس Royal Blue)

| کاربرد           | رنگ پیشنهادی             | توضیح                                   |
| ---------------- | ------------------------ | --------------------------------------- |
| Primary          | `#3d75ec` (500)          | رنگ اصلی برند و اکشن‌ها (دکمه، لینک‌ها) |
| Primary Light    | `#98bff8` (300)          | هاور یا پس‌زمینه‌های لطیف               |
| Primary Dark     | `#2648cf` (700)          | برای متون اصلی روی پس‌زمینه روشن        |
| Background Light | `rgba(255,255,255,0.05)` | برای شیشه‌ای کردن بک‌ها                 |
| Text on Light    | `#1a2451` (950)          | متن تیره برای پس‌زمینه روشن             |
| Text on Dark     | `#ffffff`                | متن روشن برای پس‌زمینه تیره             |
| Accent           | `#689ef2` (400)          | برای جزئیات، hover یا فوکوس‌ها          |

---

### 3. 📐 فاصله‌ها و گرایدها

| مورد              | مقدار             | توضیح                                       |
| ----------------- | ----------------- | ------------------------------------------- |
| Radius            | `1.5rem`          | گردی گوشه‌ها (برای شیشه‌ای بودن مهمه)       |
| Padding داخل کارت | `1.5rem` - `2rem` | فضای تنفس المان‌ها                          |
| Gap بین اجزا      | `1rem`            | فضای بین کامپوننت‌ها                        |
| Layout Grid       | `12-column grid`  | واکنش‌گرا با فاصله مناسب در دسکتاپ و موبایل |

---

### 4. 📦 کامپوننت‌ها (کلاس‌ها و ویژگی‌ها)

#### 📄 کارت شیشه‌ای

```html
<div class="glass-card">محتوا</div>
```

```css
.glass-card {
  background: rgba(255, 255, 255, 0.05);
  backdrop-filter: blur(20px);
  -webkit-backdrop-filter: blur(20px);
  border-radius: 1.5rem;
  box-shadow: 0 8px 32px rgba(0, 0, 0, 0.2);
  color: #ffffff;
  padding: 2rem;
}
```

#### 🔘 دکمه شیشه‌ای

```html
<button class="btn-glass">ارسال</button>
```

```css
.btn-glass {
  background: rgba(61, 117, 236, 0.15);
  backdrop-filter: blur(12px);
  -webkit-backdrop-filter: blur(12px);
  border-radius: 1rem;
  box-shadow: 0 4px 20px rgba(61, 117, 236, 0.25);
  color: white;
  padding: 0.75rem 2rem;
  transition: all 0.3s ease;
}
.btn-glass:hover {
  background: rgba(61, 117, 236, 0.25);
}
```

#### 💬 فرم ورودی شیشه‌ای

```css
input, textarea {
  background: rgba(255,255,255,0.08);
  backdrop-filter: blur(10px);
  -webkit-backdrop-filter: blur(10px);
  border: none;
  border-radius: 1rem;
  padding: 1rem;
  color: white;
}
input::placeholder {
  color: rgba(255, 255, 255, 0.6);
}
```

---

### 5. 📱 واکنش‌گرایی (Responsive)

* از **Flexbox و Grid** استفاده کن
* تمام کامپوننت‌ها باید روی گوشی هم با افکت شیشه‌ای، بدون اسکرول افقی کار کنند
* ‌از `@media (max-width: 768px)` برای تنظیم مجدد اندازه‌ها استفاده کن

---

### 6. 🌓 دارک مود (اختیاری)

در حالت دارک‌مود، فقط شفافیت کمتر و رنگ متون سفید باقی بمونه:

```css
body.dark {
  background: linear-gradient(135deg, #1a2451, #233785);
  color: white;
}
.glass-card, .btn-glass {
  background: rgba(0, 0, 0, 0.2);
  color: white;
}
```
